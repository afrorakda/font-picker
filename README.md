# Font Picker

Part of the Fridge Combo project. A minimalist "Font Gacha" to find the perfect style for your text in seconds.

## 1 Tool / 1 Action

Compare font styles instantly. **Font Picker** lets you cycle through a curated list of web-safe and Google Fonts using simple "Next/Back" controls. No more browsing endless font lists.

## Philosophy
* **1 Tool, 1 Action.**
* **No Login / No Ads / No Tracking**
* **Browser-only Privacy**

## How to Use
1. **Input:** Type your custom text in the input area, or use the default text.
2. **Cycle:** Tap [ ← ] or [ Pick Next Font → ] to instantly change the display font.
3. **Copy:** Tap the font name tag at the bottom of the display to copy it to your clipboard.
4. **Clear:** Use [ Clr All ] to quickly empty the input field and test new phrases.
